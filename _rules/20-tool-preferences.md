# Tool Preferences

## Editor

- Prefer `vim` for quick file edits when available

## Search and Navigation

When available, prefer these over basic alternatives:
- `rg` (ripgrep) over `grep` for content search
- `fd` over `find` for finding files
- `bat` over `cat` for syntax-highlighted viewing
- `fzf` for interactive selection
- `jq` for JSON processing

## Git

- Use `git status` before operations
- Prefer `git switch` over `git checkout` for branches
- Use `git restore` for file restoration
- Check for uncommitted changes before switching branches

## Shell

- Assume `bash` or `zsh` compatibility
- Use `set -e` in scripts to fail on errors
- Quote variables: `"$variable"`
